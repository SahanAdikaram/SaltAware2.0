<h1>SaltAware</h1>

SaltAware is a web application designed to help people with high blood pressure discover tasty, low-salt meal options using the ingredients they already have at home. Users can set their daily sodium limit, enter available ingredients, and receive personalized recipe suggestions tailored to their health needs.

The current version focuses on recipe discovery and personalization. Step-by-step cooking instructions are planned as a future enhancement.
